# Contributing

Thank you for your interest, but **this repository does not accept external
contributions**.

This is a proprietary TechX Maestro product (HCC Startpage). It is derived
from the [catppuccin-startpage](https://github.com/VolodymyrPivoshenko/catppuccin-startpage)
project but ships with HCC-specific branding, widgets, and modifications
that are **not open source**. Source is published for reference, audit, and
evaluation only. Pull requests, patches, and code contributions from
outside TechX Maestro will be closed without review.

## Upstream contributions

If you have an improvement that belongs in the upstream MIT-licensed
catppuccin-startpage codebase rather than HCC-specific additions, please
contribute it to the upstream project at
<https://github.com/VolodymyrPivoshenko/catppuccin-startpage>, not here.

## If you've found a bug

Please open a GitHub issue with a clear description and steps to reproduce.
Bug reports are welcome; code changes are not.

## If you want to report a security issue

Please follow responsible disclosure via [SECURITY.md](SECURITY.md). Do not
open public issues for security vulnerabilities.

## If you want to license, partner, or re-use

Contact **TechX Maestro** directly &mdash; <https://techxmaestro.com>.

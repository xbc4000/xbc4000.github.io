### Taking the screenshots

- Full-viewport, no browser chrome (F11 fullscreen in Firefox)
- Wait for all widgets to load (~3–5s after load) — NOW PLAYING needs a track, CONDITIONS needs geolocation, SERINA needs the `/ollama` proxy
- Save as PNG, 1920×1080 or native viewport
- `01-hero.png` is the most important — it's the one the top of the README uses as the banner
